1. Introducción y Primeros Pasos

StreamFlow Pro es tu centro de control para administrar cuentas de streaming. Diseñado para manejar cientos de perfiles, controlar vencimientos, registrar ventas y mantener todo sincronizado en la nube con Firebase.

Primera vez que entrás:

1
Registrate con email y contraseña, o usá Google.
2
Andá a Cargar y empezá a agregar cuentas (Netflix, Disney, HBO Max, etc.).
3
En Perfiles vas a ver todo cargado y listo para gestionar.
4
El Dashboard muestra métricas en tiempo real: totales, disponibles, próximos a vencer y ventas del día.
La app funciona 100% offline. Los cambios se guardan localmente y se sincronizan con Firebase cuando hay internet.
2. Cómo Cargar Cuentas

Hay tres modos para cargar datos:

Modo Cuenta Completa: Ideal para una cuenta nueva con todos sus perfiles. Seleccionás la plataforma y se generan automáticamente los perfiles según la capacidad estándar:

  • Netflix: 5 perfiles
  • Disney: 7 perfiles
  • HBO Max: 5 perfiles
  • Paramount: 5 perfiles
  • Prime Video: 6 perfiles
  • Spoty, YouTube, Crunchyroll: 1 perfil

Modo Individual: Para cargar un solo perfil específico. Ideal para renovaciones o cuentas heredadas donde solo necesitás un perfil.

Modo Masivo: Pegás un bloque de texto con varias cuentas separadas por líneas en blanco o "---". La app reconoce estos campos:

  • Mail / Email / Correo → correo electrónico
  • Contraseña / Password / Pass → clave
  • Perfil / Usuario → nombre del perfil
  • Proveedor / Vendedor → nombre del proveedor
  • Vencimiento / Fecha → fecha de vencimiento (YYYY-MM-DD)
Usá Previsualizar antes de cargar masivamente para verificar que todo esté correcto.
3. Gestión de Perfiles

Cada perfil mostrado en la lista tiene botones de acción rápida:

  • Mostrar/Ocultar contraseña: Clic en el ícono del ojo para ver la clave.
  • WA Proveedor: Abre WhatsApp del proveedor para consultas rápidas.
  • WA Cliente: Si el perfil está vendido, abre el chat con el cliente.
  • Copiar: Copia al portapapeles el mensaje con los datos del perfil para enviar al proveedor.
  • Pass: Cambia la contraseña del perfil. Pide confirmación doble por seguridad.
  • Editar: Modifica todos los campos del perfil.
  • Renovar: Extiende la fecha de vencimiento 1 mes hacia adelante. Copia automáticamente el mensaje para el cliente.
  • Libre: Quita la fecha de vencimiento. El perfil queda disponible para asignar a un nuevo cliente.
  • Duplicar: Crea una copia exacta del perfil en estado Libre.
  • Vender: Abre el modal de venta para registrar precio y WhatsApp del cliente.
  • Reasignar: Cambia manualmente la fecha de vencimiento a cualquier día.
  • Eliminar: Borra el perfil permanentemente (no se puede deshacer).

Estados de los perfiles:

Activo: Fecha de vencimiento a más de 3 días.
Próximo: Vence en 3 días o menos.
Vencido: Ya pasó la fecha de vencimiento.
Libre: Sin fecha asignada, está disponible para vender o asignar.
Vendido/En curso: Cuando se realiza la venta, el perfil mantiene sus datos pero queda marcado como vendido y no se puede volver a vender.

4. Ventas e Historial

Cómo vender un perfil:

1
En la lista de perfiles, buscá uno que esté Libre.
2
Clic en el botón Vender.
3
Ingresá el precio (opcional) y el WhatsApp del cliente (recomendado).
4
Confirmá. La app registra la venta, genera la plantilla automática y te redirige a WhatsApp para enviar el mensaje al cliente.

En la pestaña Historial podés ver, filtrar y buscar todas las ventas.

5. Dashboard y Estadísticas

El Dashboard muestra: Total, Disponibles, Próximos, Ventas hoy, Tendencias y Combos del día.

En Stats encontrás gráficos detallados por plataforma, proveedor y ganancias.

6. Filtros y Búsqueda Avanzada

En Perfiles filtrás por plataforma, estado, texto, proveedor y rango de fechas. El botón Ordenar alterna ascendente/descendente.

7. Backup y Seguridad de Datos

Exportá tus datos en formato JSON desde Perfiles. Importá para restaurar. Recomendación: backup semanal.

8. Configuración y Ajustes

En Ajustes configurás tema, sincronización, notificaciones, modo kiosco, días para "Próximo" y tamaño de letra.

9. Sistema de Token y Suscripción

Período de prueba: 3 días gratis al registrarte. El badge en el header muestra DEMO y los días restantes.

Token de suscripción: 30 días por token. Cada token funciona una sola vez y queda asociado a tu cuenta.

El token se oculta con asteriscos al escribirlo. Al validarlo, el campo se limpia automáticamente.

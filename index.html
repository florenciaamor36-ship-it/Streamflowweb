📌 1. Introducción

StreamFlow Pro es tu centro de control para administrar cuentas de streaming de manera profesional.

1
Registrate con email y contraseña, o con Google.
2
Andá a Cargar Cuentas y empezá a agregar perfiles.
3
En Perfiles gestioná todas tus cuentas.
💡 La app funciona sin internet. Los cambios se sincronizan automáticamente cuando vuelve la conexión.
📦 2. Cargar Cuentas

Tres modos de carga:

  • Cuenta Completa: Genera automáticamente todos los perfiles de una plataforma.
  • Perfil Individual: Para agregar un solo perfil específico.
  • Carga Masiva: Pegá un bloque de texto con varias cuentas.
Usá Previsualizar antes de cargar masivamente para evitar errores.
⚙️ 3. Gestión de Perfiles

Botones de acción rápida:

  • 👁️ Mostrar/ocultar contraseña
  • 📱 WhatsApp Proveedor
  • 📋 Copiar datos
  • ✏️ Editar perfil
  • 🔄 Renovar (+1 mes)
  • ⭕ Marcar como libre
  • 🗐 Duplicar perfil
  • 🏷️ Vender perfil
  • 📅 Reasignar fecha
  • 🗑️ Eliminar perfil
💰 4. Ventas

Pasos para vender un perfil:

1
Buscá un perfil en estado Libre.
2
Hacé clic en Vender.
3
Completá el precio y el WhatsApp del cliente.
4
Confirmá. La app genera la plantilla y abre WhatsApp.
📊 5. Dashboard

El Dashboard muestra métricas clave: total de perfiles, disponibles, próximos a vencer y ventas del día, además de tendencias y combos promocionales.

🔍 6. Filtros

Podés filtrar perfiles por plataforma, estado, proveedor, rango de fechas o búsqueda por texto. También podés ordenar por fecha de vencimiento.

💾 7. Backup

Exportá tus datos en formato JSON o CSV desde la pestaña Perfiles. Importá un archivo JSON para restaurar una copia de seguridad.

⚙️ 8. Configuración

En la pestaña Ajustes podés cambiar el tema, activar/desactivar sincronización automática, notificaciones, modo kiosco y ajustar el tamaño de letra.

🔑 9. Token de Suscripción

Tenés 3 días de prueba gratis. Después necesitás un token de 30 días. Contactanos por WhatsApp para adquirirlo.
